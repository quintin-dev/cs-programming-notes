## If Statement Syntax [[js/core_concepts/conditionals]]

```javascript
if (condition) {
    // code to be executed if the condition is true
}
```

## Example

```javascript
let x = 10;

if (x > 5) {
    console.log('x is greater than 5');
}
```

## If...Else Statement Syntax

```javascript
if (condition) {
    // code to be executed if the condition is true
} else {
    // code to be executed if the condition is false
}
```
